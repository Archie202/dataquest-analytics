# DataQuest Analytics

A gamified data analytics learning platform built with Next.js 16, Supabase, and React 19.

Master SQL, Python, and Excel through structured quests, projects, and boss fights. Track your XP, climb the leaderboard, and unlock achievements.

## Tech Stack

- **Frontend**: Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS
- **State**: Zustand
- **Backend**: Supabase (Auth, Database, RLS)
- **Practice Labs**: SQL execution, Python sandbox, Excel simulation

## Getting Started

```bash
npm install
cp .env.local.example .env.local   # Add your Supabase credentials
npm run dev
```

## Features

- 4 learning phases with 13 modules and 39 topics
- 144 quests (tasks, projects, boss fights)
- Interactive SQL, Python, and Excel practice labs
- XP/level system with streaks and coins
- Skill trees for SQL, Python, and Excel
- Achievement system (20 achievements to unlock)
- Leaderboard with real-time rankings
- Daily quests with progress tracking
- Theme toggle (dark/light)
